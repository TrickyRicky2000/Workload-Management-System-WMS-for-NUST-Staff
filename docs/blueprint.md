# **App Name**: UniWorkload

## Core Features:

- Role Dashboards: Separate dashboards for HOD, Academic Staff, Admin, and Supervisor with role-specific views and permissions.
- Workload Assignment: HODs can assign teaching and admin workloads (courses, hours, student count, duties) to academic staff using a standard form.
- Research Student Management: Academic Staff can add and remove their research students.
- Admin Access: Admins can view and manage all users, workloads, and system settings. They can also generate basic reports.
- Supervisor Access: Supervisors have read-only access to view all departmental workloads.
- Authentication & Roles: Secure login using Firebase Auth with role-based access control.

## Style Guidelines:

- Primary color: Calming light blue (#72BCD4), to promote focus and productivity. Derived from concepts of professionalism and knowledge work.
- Background color: Off-white (#F0F4F7), providing a clean and neutral backdrop.
- Accent color: Muted teal (#4CAF50), used sparingly for interactive elements.
- Clean and readable sans-serif font.
- Responsive layout adapts to different screen sizes.
- Simple, professional icons.
